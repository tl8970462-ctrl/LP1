## 🚀 Quick Start

1. **Extract the template**
   ```bash
   npm install
   npm run dev
   ```

2. **Customize your copy**
   Edit `lib/data.ts` and components

3. **Change your colors**
   Update `tailwind.config.ts`

4. **Connect your email service**
   Follow integration guide in CUSTOMIZATION_GUIDE.md

5. **Deploy**
   Deploy to Vercel in one click
